# 比較演算子

Pythonの比較演算子は、2つの値を比較し、ブール値（TrueまたはFalse）を返します。以下は、Pythonで使用される主な比較演算子の一覧です。

- `==` (等しい): 2つの値が等しいかどうかを比較します。
  ```python
  result = (3 == 3)  # 結果はTrue
  ```

- `!=` (等しくない): 2つの値が等しくないかどうかを比較します。
  ```python
  result = (3 != 2)  # 結果はTrue
  ```

- `>` (大きい): 左側の値が右側の値より大きいかどうかを比較します。
  ```python
  result = (5 > 3)  # 結果はTrue
  ```

- `<` (小さい): 左側の値が右側の値より小さいかどうかを比較します。
  ```python
  result = (2 < 3)  # 結果はTrue
  ```

- `>=` (大きいか等しい): 左側の値が右側の値より大きいか、または等しいかどうかを比較します。
  ```python
  result = (3 >= 3)  # 結果はTrue
  ```

- `<=` (小さいか等しい): 左側の値が右側の値より小さいか、または等しいかどうかを比較します。
  ```python
  result = (2 <= 3)  # 結果はTrue
  ```

これらの演算子を使用することで、条件分岐やループの制御など、さまざまな場面で値の比較を行うことができます。
