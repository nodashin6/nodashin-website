# 算術演算子

Pythonの算術演算子は、基本的な数学演算を行うために使用されます。以下は、Pythonで使用される主な算術演算子の一覧です。

- `+` (加算): 2つの数値を加算します。
  ```python
  result = 3 + 2  # 結果は5
  ```

- `-` (減算): 2つの数値を減算します。
  ```python
  result = 5 - 3  # 結果は2
  ```

- `*` (乗算): 2つの数値を乗算します。
  ```python
  result = 4 * 3  # 結果は12
  ```

- `/` (除算): 2つの数値を除算します。
  ```python
  result = 10 / 2  # 結果は5.0
  ```

- `//` (整数除算): 2つの数値を整数で除算し、商の整数部分を返します。
  ```python
  result = 10 // 3  # 結果は3
  ```

- `%` (剰余): 2つの数値を除算し、余りを返します。
  ```python
  result = 10 % 3  # 結果は1
  ```

- `**` (べき乗): 1つの数値を他の数値でべき乗します。
  ```python
  result = 2 ** 3  # 結果は8
  ```

これらの演算子を使用することで、Pythonでの基本的な数学演算を簡単に行うことができます。
